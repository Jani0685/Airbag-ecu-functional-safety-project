# Airbag ECU Functional Safety Project

## Project Overview
This repository contains the complete project portfolio for the design and safety analysis of an Airbag Electronic Control Unit (ECU).

## Team Structure
The project is divided into the following teams:

1. Project Management  
2. Safety Analysis  
3. System Architecture  
4. Software Development  
5. Hardware Concept Design  
6. Testing & Validation  
7. Final Documentation  

## Repository Structure
Each team must upload their work into their assigned folder.  

## Submission Rules
- Upload files only in your assigned folder  
- Use clear file names (e.g., Safety_Report_v1.pdf)  
- Do not modify other team folders  

## Timeline
Project work follows the planned Gantt chart (Week 1 – Week 7).

All contributions will be compiled into a final report located in:
